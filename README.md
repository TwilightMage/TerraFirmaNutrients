# What the hell is this?
This mod is a TFC addon that allows other mods to add various nutritional needs for players.

# Features
- set per-player nutritional needs (make a dragon that eats coal and redstone)
- change or even remove what player needs to drink (make a vampire that needs blood)
- allow player to eat rotten food (make a racoon that loves to eat trash)
